# next-ui-chakra

A design repository for building and archiving UI elements using Chakra UI.

## 🛠️ Tech Stack

- **🌐 Framework:** Next.js
- **🧠 Language:** TypeScript
- **📦 Package Manager:** Yarn
- **🖥️ Runtime:** Node.js

## 🧩 Tooling

- **📝 Form Handling:** react-hook-form
- **🎨 Styling:** Tailwind CSS
- **🧹 Linting:** ESLint
- **🧼 Formatting:** Prettier
- **🔍 Spell Checking:** cspell
